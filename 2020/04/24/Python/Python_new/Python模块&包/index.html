Python模块&包

发表于 2020-04-24 更新于 2021-05-19 分类于 Python

模块是工具的集合, 包是模块的集合

模块的概念:

导入模块:

  • import 模块名

    直接导入模块 (会导入模块中的全部工具)

    使用: 模块名.工具名

  • import 模块名 as 别名

    直接导入模块,并给模块起别名

    使用: 别名.工具名

  • from 模块名 import 工具名

    从模块中导入指定工具

    使用: 工具名

  • from 模块名 import * (不推荐使用, 导入多个模块时, 无法提示重名情况)

    从模块中导入全部工具

    使用: 工具名

模块的搜索顺序:

  1. 会根据模块名, 先在 当前目录 找指定文件, 如果有则直接导入
  2. 如果当前目录没有, 会到 系统目录 中搜索

给文件起名时, 不要与系统模块重名

模块名.__file__ (内置属性) 可以查看模块的完整路径

__name__ 属性:

__name__ 属性是Python的一个内置属性, 其中记录着一个字符串

字符串会有两种值:

  • 如果是当前执行的程序: __name__ == "__main__"
  • 如果是被其他文件导入: __name__ == 模块名

通常用法:

安装第三方模块

pip install 模块名

Python包

概念：

  • 包是一个包含多个模块的特殊目录
  • 包目录中必须有一个文件：__init__.py
  • 包名的命名方式和变量名一致

包相较于模块的优势：

可以使用 import 包名 一次性导入包中的所有模块

__init__.py 文件：

在 __init__.py 文件中指定要对外界提供的模块的列表

写法：

外界使用包：
