nginx

发表于 2021-08-05 更新于 2021-09-02 分类于 Java EE 阅读次数：

nginx 是一个轻量级、高性能的 HTTP 和反向代理 web 服务器，同时也提供了 IMAP / POP3 / SMTP 服务

nginx 简介

nginx：engine x 的简写

主要功能：

  • 负载均衡
  • 反向代理
  • 动静分离

Linux 中安装 nginx

  1. 下载 nginx：http://nginx.org/en/download.html

  2. 把下载的压缩包 nginx-1.21.1.tar.gz 放到 /usr/local/ 目录下，然后解压

  3. 进到解压后的 nginx-1.21.1 目录中，执行命令 ./configure --prefix=/usr/local/nginx-1.21.1

  4. 执行 make 命令，然后执行 nake install 命令，就安装完成了

进入 nginx-1.21.1/sbin 目录，执行 ./nginx 开启服务

开启服务的时候遇到一个报错

是因为在 nginx-1.21.1 目录下没有 logs 目录，新建一个就好了

nginx 使用

基本命令：（需要进入到 nginx-1.21.1/sbin 目录下，使用 ./nginx ... 来执行命令）

  • nginx：开启服务
  • nginx -s reload：重新加载配置文件
  • nginx -t：检查配置文件是否有错误
  • nginx -s reopen：重启服务
  • nginx -s quit：停止服务
  • nginx -s stop：强制停止服务

nginx 配置文件：

位于 nginx-1.21.1/conf/nginx.conf

使用 nginx 主要就是修改配置文件中的各种配置，从而实现不同的功能

使用 nginx 代理的 vue 应用，页面刷新就变成 404，解决方案：

在 nginx 配置文件的 location 中添加 try_files $uri $uri/ /index.html;
