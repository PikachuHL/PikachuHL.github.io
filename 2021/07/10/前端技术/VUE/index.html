Vue

发表于 2021-07-10 更新于 2021-08-03 分类于 前端技术 阅读次数：

Vue 是一套用于构建用户界面的渐进式 JavaScript 框架

Vue 基本概念

Vue 基本特点：

  • 采用组件化模式，提高代码的复用率，且让代码更好维护。
  • 声明式编码，让开发人员无需操作 DOM，提高开发效率。
  • 使用虚拟 DOM + 优秀的 Diff 算法，尽量复用 DOM 节点

初识 Vue：

  1. 想让 Vue 工作，必须创建一个 Vue 实例，且要传入一个配置对象
  2. 容器中的代码被称为 Vue 模板
  3. Vue 实例和容器是一一对应的
  4. 真实开发中只有一个 Vue 实例，且配合组件一起使用

Vue 中的 MVVM 模型

M（Model）：data 中的数据

V（View）：模板代码

VM（ViewModel）：Vue 实例

data 中所有的属性，经 Vue 处理之后，都会出现在 Vue 实例中

Vue 实例的所有属性及其原型上的所有属性，在模板中都可以直接使用

Vue 中的数据代理

JavaScript 中数据代理的基本原理：

通过 Object.defineProperty() 中的 getter 和 setter 来实现数据代理

Vue 通过数据代理，把 data 中的所有属性都拿到了 vm 中，变成了 vm 的属性，我们通过操作 vm 中的属性就可以操作 data 中的数据

而 data 在 vm 中是以 _data 属性存在的

Vue 语法

插值语法：

  • 功能：用于解析标签体内容
  • 写法：，xxx 是 js 表达式，且可以直接读取到 data 中的所有属性

指令语法：

  • 功能：用于解析标签（标签属性，标签体，绑定事件等）

  • 写法：v-xxx，

  • 例：v-bind:href="xxxx"，xxxx 同样是 js 表达式，且可以直接读取到 data 中的所有属性

Vue 数据绑定

Vue 中数据绑定分为两种：

  • 单向数据绑定
  • 双向数据绑定

单向数据绑定：数据只能从 data 流向页面

双向数据绑定：数据可以从 data 流向页面，也可以从页面流向 data

  • 指令：v-model

  • 说明：只能应用在表单类元素上，因为 v-model 操作的就是标签中 value 属性的值，可以简写为 v-model
