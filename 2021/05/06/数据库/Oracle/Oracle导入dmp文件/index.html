CDB 和 PDB：

CDB：Container DataBase（数据库容器）

PDB：Pluggable DataBase（可插拔数据库）

Oracle 12c 版本引入了 Multitenant Architecture（多租户体系结构），即一个 CDB 可以容纳多个 PDB。

在 Oracle 12c 之前，一个数据库实例只能与一个数据库相关联，一个数据库可以被多个数据库实例加载。

在 Oracle 12c 之后，一个数据库实例可以与多个数据库相关联，实例和数据库可以是一对多的关系。
