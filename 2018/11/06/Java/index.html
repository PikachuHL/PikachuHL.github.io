类与对象

构造函数：如果一个成员函数的名字与类的名字完全相同，那么在创建这个类的每一个对象时都会自动调用这个函数。这个函数就叫做构造函数。
构造函数不能有任何的返回类型。

成员变量和成员函数

成员变量 和 本地变量的区别（主要是在生存期和作用域，以及初始化问题）
本地变量是定义在函数内部的变量，
本地变量的作用域和生存期都是在函数的内部，
成员变量的作用域是类内部的成员函数，生存期是对象的生存期，

this的概念（存在于哪里）
this是连接对象和成员函数的一个对象变量。
构造函数中的 this 用法，
this.price = price
