发表于 2019-05-30 更新于 2020-05-26

AJAX：Asynchronous JavaScript and XML

JSON：JavaScript Object Notation

AJAX

概念：AJAX：Asynchronous JavaScript and XML： 异步的 JavaScript和XML

浏览器和服务器通信中的同步和异步：

  • 同步：浏览器发出请求后必须等待服务器的响应，在等待期间不能做任何操作
  • 异步： 浏览器不需要等待服务器的响应，在等待期间可以进行其他操作

AJAX是一种在无需重新加载整个网页的情况下，能够更新部分网页的技术

使用jQuery实现AJAX

  1. $.ajax()

    语法：$.ajax({键值对});

  1. $.get()

    语法：$.get(url, [data], [callback], [type]);

    • url：请求路径
    • data：请求参数
    • callback：回调函数
    • type：响应数据类型

  1. $.post()

    语法：$.get(url, [data], [callback], [type]);

JSON

概念：JavaScript Object Notation： JavaScript对象表示法

JSON 是存储和交换文本信息的语法，类似于XML

JSON 比XML更小，更快，更易解析

JSON的语法

JSON定义：

var p = {"name":"jack","age":22,"gender":true}

基本规则：

  • 数据是存储在键值对中
    • 键用引号(单双都行)引起来，也可以不使用引号
    • 值的类型：
      1. 数字（整数或浮点数）
      2. 字符串（在双引号中）
      3. 逻辑值（true 或 false）
      4. 数组（在方括号中）
      5. 对象（在花括号中）
      6. null
  • 多个键值对由逗号分隔
  • 使用 {} 定义JSON

获取 JSON 中的数据

  1. JSON对象.键名
  2. JSON对象[“键名”]

JSON 数据和Java对象的相互转换

  • JSON 解析器：

    • Jsonlib：官方提供
    • Gson：谷歌提供
    • fastjson：阿里提供
    • Jackson：SpringMVC内置

  • JSON 转为Java对象

    1. 导入 Jackson 的相关jar包

    2. 创建 Jackson 核心对象 ObjectMapper

    3. 调用ObjectMapper的相关方法进行转换

      1. readValue( JSON 字符串数据,Class)

  • Java对象转换JSON（常用）

    1. 导入 Jackson 的相关jar包
    2. 创建 Jackson 核心对象 ObjectMapper
    3. 调用ObjectMapper的相关方法进行转换
      1. 转换方法：
        • writeValue(参数1，obj) 根据参数1 类型不同有三种重载
          参数1：
    File：将obj对象转换为 JSON 字符串，并保存到指定的文件中
    Writer：将obj对象转换为 JSON 字符串，并将 JSON 数据填充到字符输出流中
    OutputStream：将obj对象转换为JSON字符串，并将 JSON 数据填充到字节输出流中

    • writeValueAsString(obj)：将对象转为 JSON 字符串

      1. 注解：

        1. @JsonIgnore：排除属性。
        2. @JsonFormat：属性值的格式化
        • @JsonFormat(pattern = "yyyy-MM-dd")

      2. 复杂 Java 对象转换

        1. List：转换后为数组
        2. Map：转换后对象格式一致
