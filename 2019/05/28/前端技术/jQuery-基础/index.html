jQuery基础

发表于 2019-05-28 更新于 2021-07-28 阅读次数：

jQuery 是 JavaScript 的一个框架，可以简化 js 代码的书写

jQuery 对象和 js 对象的区别和转换

  1. jQuery 对象在操作时，更加方便

  2. jQuery 对象和 js 对象方法不通用

  3. jQuery 对象和 js 对象的相互转换

    1. jQuery –> js ：jQuery对象[索引] / jQuery对象.get(索引)
    2. js –> jQuery ：$(js对象)

jQuery的选择器

  1. 事件绑定
    1. 规则：$("#div1").click(function(){ alert(123) });
  2. 入口函数
    1. 规则：$(function(){ 代码 });
    2. 与js 中window.onload的区别
      1. window.onload 只能定义一次，如果定义多次，后面的会把前面的覆盖掉
      2. $(function(){ 代码 }); 可以定义多次，
  3. 样式控制
    1. $("#div1").css("background-color", "pink");
    2. $("#div1").css("backgroundColor", "pink");：推荐使用

  1. 基本选择器

    1. 标签选择器（元素选择器）
    * 语法：`$("标签")` 获得所有匹配标签名称的元素
    1. id选择器
      • 语法：$("#id")获得与指定id属性值匹配的元素
    2. 类选择器
      • 语法：$(".class属性的值")获得与指定的class属性值匹配的元素
    3. 并集选择器：
      • 语法：$("选择器1,选择器2....") 获取多个选择器选中的所有元素

  2. 层级选择器

    1. 后代选择器
    * 语法：`$("A B")` 选择A元素内部的所有B元素
    1. 子选择器
      • 语法：$("A > B") 选择A元素内部的所有B子元素

  3. 属性选择器

    1. 属性名称选择器 
    * 语法：`$("A[属性名]")` 包含指定属性的选择器
    1. 属性选择器
      • 语法：$("A[属性名='值']") 包含指定属性等于指定值的选择器
      • $("A[属性名!='值']") ：属性不等于某些值，以及不包括该属性
      • $("A[属性名^='值']") ：属性的值以 ** 开头
      • $("A[属性名$='值']")：属性的值以 ** 结束
      • $("A[属性名*='值']") ：属性的值含有 **
    2. 复合属性选择器
      • 语法：$("A[属性名='值'][]...") 包含多个属性条件的选择器

  4. 过滤选择器

    1. 首元素选择器 
    * 语法： `$("A:first")`  获得选择的元素中的第一个元素
    1. 尾元素选择器
      • 语法： $("A:last") 获得选择的元素中的最后一个元素
    2. 非元素选择器
      • 语法： $("A:not(content)") 不包括指定内容的元素
      • content：选择器表达式
    3. 偶数选择器
      • 语法： $("A:even") 偶数，从 0 开始计数
    4. 奇数选择器
      • 语法： $("A:odd") 奇数，从 0 开始计数
    5. 等于索引选择器
      • 语法： $("A:eq(index)") 指定索引元素
    6. 大于索引选择器
      • 语法： $("A:gt(index)") 大于指定索引元素
    7. 小于索引选择器
      • 语法： $("A:lt(index)") 小于指定索引元素
    8. 标题选择器
      • 语法： $(":header") 获得标题（h1~h6）元素，固定写法

  5. 表单过滤选择器

    1. 可用元素选择器 
      * 语法： `:enabled`获得可用元素
    2. 不可用元素选择器
      • 语法：:disabled 获得不可用元素
    3. 选中选择器
      • 语法：:checked 获得单选/复选框选中的元素
    4. 选中选择器
      • 语法：:selected 获得下拉框选中的元素

DOM操作

内容操作

  1. html() –> 获取/设置元素的标签体内容
  2. text() –> 获取元素的标签体纯文本内容，设置元素的标签体内容
  3. val() –> 获取/设置元素的value属性值

属性操作

  1. 通用属性操作

    1. attr() –> 获取/设置自定义元素的属性
    2. removeAttr() –> 删除属性
    3. prop() –> 获取/设置固有元素的属性
    4. removeProp() –> 删除属性

  2. 对 class 属性的操作

    1. addClass() –> 添加 class 属性的值
    2. removeClass() –> 删除 class 属性的值
    3. toggleClass() –> 切换 class 属性

CRUD操作

  1. append():父元素将子元素追加到末尾
    • 对象1.append(对象2)：将对象 2 添加到对象 1 元素内部，并且在末尾
  2. prepend():父元素将子元素追加到开头
    • 对象1.prepend(对象2)：将对象 2 添加到对象 1 元素内部，并且在开头
  3. appendTo():
    • 对象1.appendTo(对象2)：将对象 1 添加到对象 2 内部，并且在末尾
  4. prependTo()：
    • 对象1.prependTo(对象2)：将对象 1 添加到对象 2 内部，并且在开头

  1. after():添加元素到元素后边

    * `对象1.after(对象2)`： 将对象 2 添加到对象 1 后边。对象 1 和对象 2 是兄弟关系

  2. before():添加元素到元素前边

    • 对象1.before(对象2)： 将对象 2 添加到对象 1 前边。对象 1 和对象 2 是兄弟关系

  3. insertAfter()

    • 对象1.insertAfter(对象2)：将对象 1 添加到对象 2 后边。对象 1 和对象 2 是兄弟关系

  4. insertBefore()

    • 对象1.insertBefore(对象2)： 将对象 1 添加到对象 2 前边。对象 1 和对象 2 是兄弟关系

  1. remove():移除元素

    • 对象.remove():将对象删除掉

  2. empty():清空元素的所有后代元素。

    • 对象.empty():将对象的后代元素全部清空，但是保留当前对象以及其属性节点
