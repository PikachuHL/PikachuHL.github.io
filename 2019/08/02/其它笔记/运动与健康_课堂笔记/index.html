运动与健康

影响人类健康的三大问题：

  • 肥胖
  • 缺乏运动导致的文明病
  • 危险与紧张带来的心理病

肥胖的危害：

血脂异常、增加脑血管病变、增加患高血压的概率、增加心脏负荷、增加糖尿病风险、易患癌症

判断肥胖的方法：

  1. BMI = w(体重_kg) / h2(身高_m)

  2. 腰围（可以测出脂肪在体内的大致分布）

    • 世界卫生组织：男 < 95cm，女 < 80cm
    • 中国：男 < 85cm，女 < 80cm

  3. 腰臀比：男 < 0.95，女 < 0.85 向心性肥胖

  4. 标准体重_kg = 身高_cm - 105

肥胖是仅次于吸烟和艾滋病的第三大慢性杀手

运动是减少脂肪最经济有效的方法

判断心理健康的四个标准：

  1. 心理标准
  2. 心理测验标准
  3. 统计标准
  4. 社会适应标准

运动的好处：

  • 健康幸福感
  • 对抑郁有治疗作用
  • 产生良好的情绪体验
  • 对意志品质有促进作用
  • 使认知能力提高

生命在于科学运动

科学运动的关键：

  • 确定运动项目
  • 设定运动量与运动强度
  • 把握运动时间

有氧运动和无氧运动：

  • 有氧运动：中低强度、时间长、不间断、有节奏（走、慢跑、游泳、健身操）
  • 无氧运动：强度高、时间短、疲劳难以消除（短跑、举重、拳击）

靶心率 =（220-年龄）*65%(下限) 或 85%(上限)

运动量 = 运动强度 * 运动时间

每周4~5次运动最为合适（贵在持之以恒）

在餐后30分钟~1小时运动最为合适

晚上6~8点运动合适
