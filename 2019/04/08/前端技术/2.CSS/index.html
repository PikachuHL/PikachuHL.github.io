CSS 主要是给 HTML “美容” 的

CSS 的基本概念

CSS（Cascading Style Sheets）：层叠样式表

作用：给 HTML 添加样式

CSS 的声明

  1. 在标签上使用 style 属性进行声明 （内联样式） – 不常用

    作用：此声明会将 css 样式直接作用于当前标签

  2. 在 head 标签中使用 style 标签声明 （内部样式）

    作用：此声明一般声明当前网页的公共样式

  3. 在 head 标签中使用 link 标签引入外部声明好的 css 文件 （外部样式）

    作用：此声明适用于不同网页间样式重复使用的问题

不同的声明给同一个标签操作了同一个样式，遵循就近原则进行显示

CSS 的选择器

  1. 标签选择器：将当前网页内的所有该标签增加相同的样式

    标签名{样式名1:样式值1;......}

  2. id 选择器：选择具体的 id 属性值的元素

    #id属性值{样式名1:样式值1;......}

  3. 类选择器：选择具有相同 class 属性值的元素

    .class属性值{样式名1:样式值1;......}

  4. 全部选择选择器：选择所有的 HTML 标签，并添加相同的样式

    *{样式名1:样式值1;......}

  1. 组合选择器：解决不同的选择器之间重复样式的问题

    选择器1,选择器2{样式名1:样式值1;......}

  2. 子标签选择器

    选择器1 子标签选择器2{样式名1:样式值1;......}

  3. 属性选择器：指定某属性为特定值的标签的样式

    标签名[属性名="属性值"]{样式名1:样式值1;......}

  4. 伪类选择器：选择一些元素具有的状态,

    元素:状态{}

CSS 的使用过程：

  1. 声明 CSS 代码域
  2. 使用选择器选择要添加样式的标签
    1. 使用 * 选择器给整个页面添加基础样式
    2. 使用类选择器给不同的标签添加基础样式
    3. 使用标签选择器给某类标签添加基础样式
    4. 使用 id 选择器、属性选择器、style 属性声明给单独标签添加样式
  3. 书写样式单

CSS 的常用样式

  • 边框设置
    • border:solid 1px;
  • 字体设置
    • font-size:10px; –> 设置字体大小
    • font-family:"宋体"; –> 设置字体格式
    • font-weight:bold; –> 加粗
  • 字体颜色设置
    • color:颜色;
  • 背景颜色设置
    • background-color:颜色;
  • 背景图片设置
    • background-image:url(图片路径); –> 设置背景图片
    • background-repeat:no-repeat; –> 设置图片不重复
    • background-size:cover; –> 图片平铺整个页面
  • 高和宽设置
    • width:200px;
    • height:300px;
  • 浮动设置
    • float:left/right;
    • 浮动设置可将列表横向显示
  • 行高(行间距)设置
    • line-height:10;

CSS 的盒子模型

div 标签：

块级标签，主要是用来进行网页布局的，会将其中的子元素内容作为一个独立的整体存在

特点：

  • 默认宽度是页面的宽度，可以设置
  • 默认没有高度，可以设置

盒子模型：

由内向外以此为：

  • 内容区域：

    • 作用：改变内容区域的大小
    • 设置宽和高即可改变内容区域的大小

  • 内边距：padding

    • 作用：设置内容和边框之间的距离
    • 注意：内边距不会改变内容区域的大小
    • 可以单独设置上下左右

  • 边框：border

    • 作用：用来设置元素边框的大小
    • 可以单独设置上下左右

  • 外边距：margin

    • 作用：用于设置元素和元素之间的间隔
    • 居中设置：margin: 0px auto; 上下间距为 0，左右居中
    • 可以单独设置上下左右

CSS 的定位

position：

  • 相对定位：relative
    • 作用：相对元素原有的位置移动指定的距离
    • 注意：其他元素的位置是不改变的
  • 绝对定位：absolute
    • 作用：相对于元素的参照界面进行移动
    • 注意：默认参照元素为界面，如果父级元素使用了相对定位，则成为参照元素
  • 固定定位：fixed
    • 作用：将元素固定在页面的指定位置

以上定位都可以用 top、bottom、left、right 来进行移动

z-index：声明元素的显示级别

hover：可用于设置与鼠标相关的动作
