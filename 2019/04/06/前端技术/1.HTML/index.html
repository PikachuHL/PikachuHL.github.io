HTML 是互联网的三大基石之一

HTML 的基本概念

1、HTML（Hyper Text Markup Language）：超文本标记语言

  • HTML 是由浏览器解析执行的

2、作用：告诉浏览器接收到的数据以何种数据组织形式进行展示

3、互联网的三大基石：

  • URL
  • HTTP
  • HTML

4、HTML 文档后缀名：.html 或 .htm

5、HTML 的标签分类：

  • 围堵标签：有开始标签和结束标签，如 <html></html>
  • 自闭和标签：开始标签和结束标签在一起，如 <br/>

6、HTML 标签不区分大小写，建议使用小写

7、在开始标签中可以定义属性。属性是由键值对构成，值需要用引号（单双都可）引起来

图片

标签：<img />

属性：

  • src="图片路径" （本地路径或URL）
  • width = "图片宽度"
  • height = "图片高度"
  • title = "图片标题" （鼠标放到图片上的时候会显示）
  • alt = "图片加载失败时显示的文字"

只修改图片的一个尺寸属性时，图片会等比例变化

超链接

标签：<a />

属性：

  • href：要跳转的网页资源路径
    • 可以是本地资源（本地 html 文件，建议填写相对路径）
    • 也可以是网络资源（URL）
  • target：指明跳转之后的网页显示的位置
    • _self –> 在当前页刷新显示
    • _blank –> 在新的标签页中打开
    • _top –> 在顶层页面中显示
    • _parent –> 在父级页面中显示

框架

标签：<iframe></iframe>

作用：在当前网页中加载其他网页的资源，达到同一个页面显示多个网页资源的目的

属性：

  • src：要显示的网页路径（本地或网络）
  • width：设置显示区域的宽度
  • height：设置显示区域的高度
  • name：设置内嵌区域的名字，结合超链接标签的 target 属性使用

frameset：可以实现 iframe 的功能，但更灵活

作用：把当前页面切分成不同的块，每一块都可以独立的显示一个网页

属性：

  • rows：按照行切分页面

  • cols：按照列切分页面

  • 子标签：frame –> 进行切分区域的占位

    • src：资源路径
    • name：区域名称，结合超链接使用

注意事项：

使用框架标签时要删除 body 标签
