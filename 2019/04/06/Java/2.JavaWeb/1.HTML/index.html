HTML 是互联网的三大基石之一

HTML

一、HTML 的基本概念

1、HTML（Hyper Text Markup Language）：超文本标记语言

  • HTML 是由浏览器解析执行的

2、作用：告诉浏览器接收到的数据以何种数据组织形式进行展示

3、互联网的三大基石：

  • URL
  • HTTP
  • HTML

4、HTML 文档后缀名：html 或 htm

5、HTML 的标签分类：

  • 围堵标签：有开始标签和结束标签，如<html> </html>
  • 自闭和标签：开始标签和结束标签在一起，如<br/>

6、HTML 标签不区分大小写，建议使用小写

7、在开始标签中可以定义属性。属性是由键值对构成，值需要用引号（单双都可）引起来

1、网页标题标签

2、编码格式标签

4、网页自动跳转标签

5、其他标签

三、body 标签–文本标签

1、标题标签

h1 到 h6：将其中的文本加粗加黑显示，自动换行（块级标签）

属性：align：center left right

2、水平线标签

hr：会在页面中显示一条水平线，默认全尺寸居中显示

属性：

  • width="宽度"
  • size="高度"
  • color="颜色"
  • align="位置"

3、段落标签

p：会将一段数据作为整体进行显示，方便 css 和 js 操作，自动换行（块级标签）

4、换行标签

br：在此位置换行

5、空格符

&nbsp ; –> 在此位置增加空格

6、权重标签

b：将内容加黑显示

i：将内容斜体显示

u：将内容加下划线

del：将内容加删除线

无序列表

有序列表

自定义列表

五、body 标签–图片标签

1、图片标签支持静态图片和动态图片

2、图片标签为行内元素，不会自动换行

3、图片路径：

  • 本地：相对路径或绝对路径，常用相对路径
  • 网络：图片 URL

4、属性：

  • width = "图片宽度"
  • height = "图片高度"
  • title = "图片标题" （鼠标放到图片上的时候会显示）
  • alt = "图片加载失败时显示的文字"

六、超链接标签

a 标签属性：

  • 超链接标签为行内元素，无自动换行功能

  • href：要跳转的网页资源路径

    • 可以是本地资源（本地 html 文件，建议填写相对路径）
    • 也可以是网络资源（URL）

  • target：指明跳转之后的网页显示的位置

    • _self –> 在当前页刷新显示
    • _blank –> 在新的标签页中打开
    • _top –> 在顶层页面中显示
    • _parent –> 在父级页面中显示

链接点击处：

  • 可以是文字
  • 也可以是图片

八、内嵌标签

iframe

作用：在当前网页中加载其他网页的资源，达到同一个页面显示多个网页资源的目的

属性：

  • src：要显示的网页路径（本地或网络）
  • width：设置显示区域的宽度
  • height：设置显示区域的高度
  • name：设置内嵌区域的名字，结合超链接标签的target属性使用

九、框架标签

frameset：可以实现 iframe 的功能，但更灵活

作用：把当前页面切分成不同的块，每一块都可以独立的显示一个网页

属性：

  • rows：按照行切分页面

  • cols：按照列切分页面

  • 子标签：frame –> 进行切分区域的占位

    • src：资源路径
    • name：区域名称，结合超链接使用

注意事项：

使用框架标签时要删除 body 标签

十、form 标签 & 表单域标签

form 标签

作用：收集并提交用户数据给指定服务器

属性：

  • action：收集到的数据的提交地址
  • method：收集数据的提交方式
    • get：适合小量数据，表单数据以?隔开写在 URL 后面，不安全
    • post：适合大量数据，安全

注意：

  • form 标签收集其内部的数据
  • 要提交的表单项必须拥有 name 属性（作为键），否则不会提交
  • 表单数据提交的是表单域标签的 value 值

form 表单域标签

form 表单域标签写在 form 标签内部
