数据结构与算法基本概念

发表于 2019-03-28 更新于 2021-08-06 阅读次数：

数据结构是计算机存储、组织数据的方式

算法是计算机解题的过程

一、数据结构

  • 逻辑结构
    • 线性结构（线性表、栈、队列、串、数组）
    • 非线性结构（树、图）
  • 物理（存储）结构
    • 顺序结构
    • 链表结构
    • 索引结构
    • 散列结构
  • 数据运算：插入、删除、修改、查询、排序

逻辑结构唯一
存储结构不唯一
运算的实现依赖与存储结构

二、算法

算法是计算机解题的过程

算法优劣的评判依据：时间复杂度和空间复杂度

时间复杂度

1、时间复杂度是表示问题的规模而不是指算法具体的运行时间或者运行次数（时间频度）
2、时间复杂度的表示：T(n) = O(f(n))
3、时间复杂度一般有最坏时间复杂度和平均时间复杂度
4、我们一般讨论的是最坏时间复杂度

5、时间复杂度的计算

  • 找出算法中的基本语句（执行次数最多的语句就是基本语句）
  • 计算基本语句执行次数的数量级
  • 将求得的数量级放入 O() 中

6、常见的时间复杂度

  • 常数阶 O(1)
  • 对数阶 O(log2n)
  • 线性阶 O(n)
  • 线性对数阶 O(n*log2n)
  • 平方阶 O(n2)
  • 立方阶 O(n3)
  • …
  • k次方阶 O(nk)
  • 指数阶 O(2n)
  • 阶乘阶 O(n!)

上面的时间复杂度越来越高，执行效率越来越低

空间复杂度

S(n) = O(g(n))
空间复杂度比时间复杂度分析要少；
在使用递归时算法的空间复杂度会比较高
